# 🏎 PITSTOP – Formula 1 Dashboard

**PITSTOP** is a modern Formula 1 fan dashboard that provides information about drivers, teams, race calendars, and championship standings. The website is designed to give F1 fans a clean and engaging platform to explore key details about the Formula 1 season in one place.

The site features dedicated pages for drivers, teams, race schedules, and standings, presented with a racing-inspired user interface. It aims to replicate the experience of professional sports dashboards while keeping the design simple, fast, and easy to navigate.

## 🚀 Features

* Driver profiles and team information
* Race calendar for the Formula 1 season
* Championship standings overview
* Dedicated pages for drivers, teams, and races
* Responsive design for different screen sizes

## 🛠 Tech Stack

* Wix Vibe (Website Builder)
* React-based page structure
* API-based data integration

## 🎯 Purpose

This project was created as a learning and portfolio project to explore web development, API integration, and sports dashboard design. The goal was to build a visually appealing and informative platform for Formula 1 fans while demonstrating modern web development practices.
